Constant epic_core::consensus::MAX_BLOCK_WEIGHT[−][src]

pub const MAX_BLOCK_WEIGHT: usize = 40_000;
Expand description

Total maximum block weight. At current sizes, this means a maximum theoretical size of:

  • (674 + 33 + 1) * (40_000 / 21) = 1_348_571 for a block with only outputs
  • (1 + 8 + 8 + 33 + 64) * (40_000 / 3) = 1_520_000 for a block with only kernels
  • (1 + 33) * 40_000 = 1_360_000 for a block with only inputs

Regardless of the relative numbers of inputs/outputs/kernels in a block the maximum block size is around 1.5MB For a block full of “average” txs (2 inputs, 2 outputs, 1 kernel) we have - (1 * 2) + (21 * 2) + (3 * 1) = 47 (weight per tx) 40_000 / 47 = 851 (txs per block)
